## 八、反推计算示例

以 **MiniMax Max ¥119** 为例：
- Token 额度：18 亿 tokens/月
- 旗舰模型 M2.7 混合单价：2.5 元/百万（保守取值）
- 等效 API 价值 = (1,800,000,000 ÷ 1,000,000) × 2.5 = **¥4,500**
- 实际折扣率 = 119 ÷ 4500 = **2.6%**
- 意思：**花 ¥119 买到了价值 ¥4,500 的 API 调用量**（约 1/38 的价格）

以 **Qwen 高级坐席 ¥698** 为例：
- Credits 额度：100,000 Credits
- 按 Qwen3.6-Plus 兑换：100,000 × 500 = 5,000 万 tokens
- Qwen3.6-Plus 混合单价：1.4 元/百万
- 等效 API 价值 = (50,000,000 ÷ 1,000,000) × 1.4 = **¥70**
- 实际折扣率 = 698 ÷ 70 = **997%**
- 意思：**Qwen 高级坐席比按量付费贵了约 10 倍**，但买的是团队管理、模型超市、无窗口限制等附加价值
